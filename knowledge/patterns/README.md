# Patterns

Learned patterns from vault analysis and research.

## Categories

- `workflow/` - User workflow patterns
- `tools/` - Tool usage patterns
- `timing/` - Temporal patterns
- `content/` - Content creation patterns

## Format

Each pattern file:
```markdown
# Pattern: [Name]

## Trigger
What signals this pattern

## Observed Behavior
What the user typically does

## Suggested Response
How brain should respond

## Confidence
[low/medium/high] based on observations

## Evidence
- Source 1
- Source 2
```
